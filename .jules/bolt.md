## 2024-05-18 - Pandas Normalization and Dict Conversion Optimization
**Learning:** `pd.json_normalize` handles nested data slowly when it needs to be serialized via `apply(lambda x: ...)`. Pre-serializing lists to JSON strings natively using Python's `json.dumps` recursively before sending the dictionaries to `json_normalize` speeds up operations and guarantees consistent outputs. Moreover, using `df.to_dict('records')` can be surprisingly slow; instead, using a list comprehension with `itertuples()` and `zip` yields a noticeable speedup (~1.3x).
**Action:** When needing to flatten a large list of dictionaries containing lists using pandas, pre-serialize those lists in native Python rather than relying on post-processing DataFrame columns. Additionally, use `[dict(zip(df.columns, row)) for row in df.itertuples(index=False, name=None)]` when converting a pandas DataFrame back to a list of dicts.
